## 📘 Descrição do Projeto
Projeto que busca demonstrar como é a Dengue no Estado de Goiás.
Foi utilizado um dataset público do estado de Goiás para produzir o material.

 
## 🛠 Tecnologias Utilizadas
- Linguagem: Python
- Ferramentas: Jupyter Notebook, pandas, matplotlib
- Dataset: SEIDIGI (https://dadosabertos.go.gov)
